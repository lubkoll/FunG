We begin with the function \[f(x)=x^{3/2}+\sin(x^{1/2}).\]

#include "fung.hh"

template <class Variable>
auto generateFunction(const Variable& x)
{
  return FunG::finalize( pow<3>(sqrt(x)) + sin(sqrt(x)) ); 
}

Usage is as follows:

auto f = generateFunction(variable<0>(1.));

f.template update<0>(1.) //  change function argument

auto value = f(); // or f.d0()
auto firstDerivative  = f.template d1<0>();
auto secondDerivative = f.template d2<0,0>();
auto thirdDerivative  = f.template d3<0,0,0>();

The call to finalize is necessary to simplify usage of the generated function. For functions depending only on scalar variables it adds default arguments for the directional derivatives. Without usage is slightly less convenient. This is illustrated in the other examples.

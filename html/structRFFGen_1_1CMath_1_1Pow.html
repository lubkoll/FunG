Detailed Description

template<int dividend, int divisor = 1>
struct RFFGen::CMath::Pow< dividend, divisor >

Power function with rational exponent including first three derivatives.

For scalar functions directional derivatives are less interesting. Incorporating this function as building block for more complex functions requires directional derivatives. These occur during applications of the chain rule. For the cases and specializations are used that avoid the use of std::pow.
